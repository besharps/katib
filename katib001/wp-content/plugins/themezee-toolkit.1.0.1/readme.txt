== Description ==

The *ThemeZee Toolkit* is a collection of useful small modules and features, neatly bundled into a single plugin. This plugin has been designed specifically for ThemeZee WordPress themes but will also work with any theme. 

= Included Features =

* Header & Footer Scripts - Insert custom code in the head and footer section of your theme
* Widget Visibility - Hide or show widgets conditionally
* Gallery Carousel - Transform your galleries into a gorgeous full-screen photo experience

More modules are coming soon!

= Documentation =

For detailed setup instructions, visit the official [Documentation](http://themezee.com/docs/toolkit-documentation/) page

== Installation ==

1. Activate the plugin
2. Go to Appearance > Theme Addons > Toolkit and enable all the modules you need
3. For detailed setup instructions, visit the official [Documentation](http://themezee.com/docs/toolkit-documentation/) page
